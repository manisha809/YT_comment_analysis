# YT_comment_analysis
## About the Project
In this Web app you can give any YouTube video's Videoid, and you will get sentiment analysis of the video based on the comments posted in that video.
Comments are classified into 3 different sentiments :
 1.Positive Comments
 2.Negative Comments
 3.Neutral Comments
